Apple Logo

Objective: Create the striped apple logo in Illustrator.

Problem: How can we get the stripes on the apple? How can we perfectly trace an apple shape?

Solution: Using the Place tool to put an image into the canvas, we can trace the apple logo. By using colored rectangles and the pathfinder tool we can turn the apple into a bunch of rectangles that make the stripes.
